# Monthly Reminder App - Design Style Guide

## Design Philosophy

### Visual Language
- **Modern Minimalism**: Clean, uncluttered interface that prioritizes content and functionality
- **Sophisticated Elegance**: Professional aesthetic suitable for both personal and business use
- **Intuitive Interaction**: Every element designed for immediate understanding and ease of use
- **Calming Productivity**: Colors and layouts that promote focus without causing stress

### Color Palette
- **Primary**: Soft sage green (#8B9A7A) - representing growth and organization
- **Secondary**: Warm cream (#F7F5F3) - providing gentle contrast and warmth
- **Accent**: Muted coral (#E8A598) - for important actions and highlights
- **Neutral**: Charcoal gray (#4A4A4A) - for text and subtle elements
- **Background**: Pure white (#FFFFFF) - maintaining clean, spacious feel

### Typography
- **Display Font**: "Tiempos Headline" - Bold serif for headings and emphasis
- **Body Font**: "Suisse Int'l" - Clean sans-serif for readability and modern feel
- **Monospace**: "SF Mono" - For time displays and technical elements

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and state transitions
- **ECharts.js**: Data visualization for productivity analytics
- **Splide.js**: Smooth carousel for reminder categories
- **p5.js**: Subtle background particle effects
- **Pixi.js**: Advanced visual effects for celebration animations

### Animation & Effects
- **Micro-interactions**: Gentle hover states and button feedback
- **Page Transitions**: Smooth slide animations between sections
- **Loading States**: Elegant skeleton screens and progress indicators
- **Success Animations**: Celebratory effects for completed tasks
- **Parallax Elements**: Subtle depth with floating UI elements

### Header & Navigation
- **Fixed Navigation**: Clean top bar with smooth scroll effects
- **Breadcrumb System**: Clear navigation hierarchy
- **Search Integration**: Prominent search with real-time filtering
- **User Profile**: Elegant avatar and settings access

### Layout & Spacing
- **Grid System**: 12-column responsive grid for perfect alignment
- **Generous Whitespace**: Breathing room for reduced cognitive load
- **Card-based Design**: Organized content in digestible containers
- **Consistent Margins**: 24px base unit for visual rhythm

### Interactive Elements
- **Button Styles**: Rounded corners with subtle shadows and hover lift
- **Form Design**: Clean inputs with floating labels and validation
- **Modal Overlays**: Elegant popups with backdrop blur effects
- **Toggle Switches**: Smooth state changes with visual feedback

### Background & Atmosphere
- **Subtle Textures**: Light paper grain for tactile feel
- **Gradient Overlays**: Gentle color transitions for depth
- **Geometric Patterns**: Minimal shapes as decorative elements
- **Lighting Effects**: Soft shadows and highlights for dimension

## Mobile Considerations
- **Touch-friendly**: 44px minimum touch targets
- **Swipe Gestures**: Intuitive mobile interactions
- **Responsive Typography**: Scalable text for all screen sizes
- **Thumb Navigation**: Bottom-aligned primary actions

## Accessibility Features
- **High Contrast**: 4.5:1 minimum contrast ratios
- **Focus Indicators**: Clear keyboard navigation
- **Screen Reader**: Semantic HTML and ARIA labels
- **Motion Preferences**: Respect reduced motion settings
