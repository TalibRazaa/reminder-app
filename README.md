# RemindMe - Monthly Reminder App

A modern, user-friendly monthly reminder application with beautiful UI and smooth animations.

## Features
- Interactive calendar with reminder management
- Dashboard with productivity analytics
- Customizable themes and settings
- Local storage persistence
- Responsive design

## Technologies Used
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- Anime.js for animations
- ECharts for data visualization
- Splide.js for carousels

## How to Run
1. Open `index.html` in a web browser
2. Or use Live Server in VS Code
